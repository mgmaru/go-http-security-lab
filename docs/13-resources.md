# 学習時に使う主な資料

### HTTP

- MDN Web Docs
- Go公式`net/http`ドキュメント
- RFC 9110

RFCは最初から通読せず、HTTPの正式な意味を確認する辞書として使用します。

### Go

- `net/http`
- `net/http/httptest`
- `database/sql`
- `context`
- Go公式チュートリアル

### APIクライアント

- `curl`
- Bruno
- 必要に応じてPostmanまたはHTTPie

APIクライアントはHTTPの理解を置き換えるものではなく、リクエストの保存・整理・再利用のために使用します。

### Webセキュリティ

- PortSwigger Web Security Academy
- Burp Suite Community Edition
- OWASP Juice Shop
- OWASP WebGoat
- OWASP Cheat Sheet Series
- IPA「安全なウェブサイトの作り方」

---

## 資料の使い方

- 最初からすべてを通読しない
- まず自分で疑問を言語化する
- 公式ドキュメントや一次資料を優先する
- RFCは必要な意味を確認する辞書として使う
- AIから得た説明は、重要な仕様について一次資料で確認する
- 完成コードを探すより、概念・仕様・制約を調べる
